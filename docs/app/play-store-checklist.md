# 구글플레이 출시 체크리스트 (TENTENS)

> 전략: **내부 테스트 트랙 먼저** → 비공개/공개 테스트 → 프로덕션. 광고(④)는 출시 이후.

## A. 계정·기본

- [ ] 구글플레이 개발자 계정 등록 ($25, 1회)
- [ ] 앱 생성: 이름 **TENTENS**, 무료, 게임
- [ ] 카테고리: **게임 → 카드** 또는 **캐주얼**
- [ ] ⚠️ 제목/설명에 **"poker" 등 도박 연상 키워드 회피** (연령등급·심사 리스크). "전략 카드 배치" 류 표현 사용.

## B. 스토어 등록정보 자산

- [ ] 앱 아이콘 **512×512 PNG** (브랜드 아이콘 — 현재 Capacitor 기본값이라 교체 필요)
- [ ] 그래픽 이미지(feature graphic) **1024×500**
- [ ] 스크린샷: 폰 최소 2장 (게임이 **가로**이므로 가로 스크린샷)
- [ ] 짧은 설명(80자), 전체 설명(4000자)

## C. 정책·양식 (필수 통과 항목)

- [ ] **개인정보처리방침 URL**: `https://www.tentens.kr/privacy` (이미 존재 — 아래 D 확인)
- [ ] **데이터 보안(Data safety) 양식**:
  - 수집: 계정(아이디/닉네임), 비밀번호(해시 저장), 게임 점수/전적. (위치·연락처·광고ID 없음 — 광고는 ④에서 재검토)
  - 전송 중 암호화(HTTPS) 예
  - **계정 삭제 경로 제공 여부** — 아래 D 참고 (구글 요구사항)
- [ ] **콘텐츠 등급(IARC 설문)**: 카드 게임. **"모의 도박(simulated gambling)" 문항을 정확히** 답할 것.
  - 실제 베팅·현금·확률형 뽑기 없음 → 도박 아님으로 답하는 게 맞지만, 포커 조합을 쓰므로 심사자가 볼 수 있음. 정직하게 응답.
- [ ] **타겟 연령·대상(Target audience)** 설정
- [ ] **앱 접근 권한(App access)**: 로그인이 필요하므로 **심사자용 테스트 계정** 제공(아이디/비번)

## D. 출시 전 코드/서비스 확인 (선행 작업)

- [x] **계정 삭제 기능** — 인앱 "계정 삭제"(로비 우상단 계정 메뉴) + `POST /api/account/delete`(비번 확인 후 삭제, FK 캐스케이드). end-to-end 검증 완료.
  - **데이터 삭제 URL**(데이터 보안 양식용): `https://www.tentens.kr` 로그인 → 계정 메뉴 → 계정 삭제. (웹뷰 앱이라 인앱=웹 동일 경로)
- [x] **앱 아이콘** — `store-assets/play-store-icon-512.png`(리스팅 512), 안드로이드 런처 아이콘 전 밀도 교체. AAB 빌드 시 자동 포함.
- [x] `/privacy` — 수집 데이터·계정 삭제 경로 명시 반영 완료.
- [ ] **#4 RLS 강화** — service_role 코드 배포됨. **아침에 재잠금 SQL 실행 필요**(0004~0006). 설계: [rls-hardening-design](../superpowers/specs/2026-07-02-rls-hardening-design.md)
- [ ] 릴리스 서명 AAB 빌드: [android-release.md](android-release.md) (Android Studio)
- [ ] 스토어 등록정보 자산(스크린샷·설명·그래픽) 준비

## E. 업로드·출시

- [ ] 내부 테스트 트랙에 AAB 업로드 → 테스터(본인/지인)로 설치 확인
- [ ] 실기기에서 로비/로그인/멀티/리더보드/친구 정상 동작
- [ ] 단계적 확대: 내부 → 비공개 → 프로덕션
