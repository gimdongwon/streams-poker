- **수집 항목**:
  - 개인정보 > 사용자 ID (username, 계정 식별용)
  - 개인정보 > 이름 (nickname, 표시 이름)
  - 앱 활동 > 인앱 활동/게임 진행(게임 점수·랭킹) — 계정에 연결
  - (분석) 앱 활동/기기 식별자 — Google Analytics 사용 시 해당
- **각 항목 목적**: 앱 기능(계정·게임·랭킹·친구), 분석
- **전송 중 암호화**: 예 (HTTPS)
- **사용자가 데이터 삭제 요청 가능**: 예
  - 삭제 방법 URL: `https://www.tentens.kr` (로그인 → 우상단 계정 메뉴 → 계정 삭제)
- 비밀번호는 단방향 해시 저장(인증용) — 별도 데이터 유형 declare 불필요, 정직하게 기재
